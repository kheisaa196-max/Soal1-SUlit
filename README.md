Program ini membantu seorang guru untuk menginput nilai ujian sejumlah siswa dan menghitung nilai rata-ratanya.

Cara kerja program:

Program meminta input jumlah siswa.

Pengguna memasukkan nilai ujian untuk setiap siswa.

Semua nilai dijumlahkan menggunakan loop for.

Setelah semua nilai dimasukkan, program menghitung rata-rata dengan rumus
Program menampilkan hasil rata-rata ke layar.
